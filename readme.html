安裝：著名的五分鐘快速安裝法

  1. 將 WordPress 壓縮檔解壓縮至一個空的目錄並全部上傳。
  2. 使用您的瀏覽器開啟 wp-admin/install.php。這將會指引您設定並產生 wp-config.php檔案，其內含資料庫相關設定。
    1. 若基於某些原因無法這麼做，別擔心。這並非每種網頁伺服器都運作正常。使用文字編輯器開啟 wp-config-sample.php檔案，填妥資料庫相關設定。
    2. 將檔案儲存為 wp-config.php 並上傳它。
    3. 使用您的瀏覽器開啟 wp-admin/install.php。
  3. 一旦您的設定檔案正確，安裝程式將為您的網誌建立資料表。若出現錯誤請仔細檢查您的 wp-config.php 檔案，並再試一次。若仍失敗，請前往 WordPress Taiwan 正體中文討論區尋求解決辦法。
  4. 若您並未輸入密碼，請記下它顯示的密碼。若您並未提供管理員帳號，它將會是 admin。
  5. 安裝程式應會將您引導至登入頁面。使用您在安裝過程中取得之帳號密碼登入。若密碼為自動產生，您可點擊「個人資料」頁面來變更密碼。
